## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+
- pnpm 10+
- MySQL/TiDB database
- Manus OAuth credentials (for deployment)

### Installation

```bash
# Clone the repository
git clone https://github.com/bekingdomcomejoker-cpu/omega-warfare-analytics.git
cd omega-warfare-analytics

# Install dependencies
pnpm install

# Set up environment variables
cp .env.example .env.local
# Edit .env.local with your database and OAuth credentials

# Run database migrations
pnpm run db:migrate

# Create Primary Researcher account
pnpm run create:researcher

# Start development server
pnpm run dev
```

### Access the Dashboard

Navigate to `http://localhost:5173` and log in with your Manus OAuth credentials.
